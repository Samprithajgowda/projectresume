ANIMATED PORTFOLIO INFOGRAPHIC

Project Overview

This is a modern interactive portfolio website built using Django, HTML,
CSS, and JavaScript.

Features

-   Clean infographic layout
-   Animated center portrait
-   Message-style information boxes
-   Typing animation on hover
-   Clickable contact links (Email, LinkedIn, GitHub)
-   Smooth hover effects and transitions

Technologies Used

-   Python
-   Django
-   HTML5
-   CSS3
-   JavaScript

Project Structure

PROJECTRESUME/ │ ├── blog/ │ ├── static/blog/style.css │ ├──
templates/blog/home.html │ ├── views.py │ ├── urls.py │ ├── blogproject/
│ ├── settings.py │ ├── manage.py

How to Run

1.  Clone the repository
2.  Navigate to the project folder
3.  Run: python manage.py runserver
4.  Open: http://127.0.0.1:8000/
